# Repository Instructions

- Always read this file before starting work in this repository.
- This project is a dependency-free static browser game. Prefer plain HTML, CSS, and JavaScript unless a feature clearly needs a build tool.
- Keep the first screen playable. Avoid marketing or landing-page scaffolding in front of the game.
- The premise is climate and celebrity-private-jet satire. Do not add live flight tracking, real-time personal location data, or unsourced claims about real people.
- Use fictionalized or clearly labeled public-estimate benchmarks for parody targets.
- Preserve the lightweight local workflow: opening `index.html` in a browser should be enough to play.
- After JavaScript changes, run `node --check app.js` before handing off.
